# Pipeline de Dados: Laudos Veterinários (dbt + Postgres)

Projeto de extração, limpeza e modelagem de dados provenientes de laudos veterinários complexos. O objetivo é transformar dados brutos em formato XML (codificados em Base64) num Data Warehouse organizado em Star Schema para análise clínica.

# Tecnologias Utilizadas
- dbt: Transformação e modelagem de dados.
- PostgreSQL: Banco de dados para armazenamento do Data Warehouse.
- SQL: Linguagem para extração de lógica XML (XPath).
- DBeaver: Gestão e visualização da base de dados.
